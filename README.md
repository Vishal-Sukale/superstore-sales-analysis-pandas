
# Superstore Sales Analysis using Pandas

## Problem Statement
Businesses generate large amounts of sales data, but extracting meaningful insights from it is challenging. This project performs Exploratory Data Analysis (EDA) on the Superstore dataset to identify sales trends, profit patterns, and key business insights.

## Objective
- Analyze sales and profit performance  
- Identify top-performing categories and regions  
- Discover trends and patterns in data  
- Provide actionable business insights  

## Dataset Description
- Dataset: Superstore Sales Dataset  
- Type: Tabular data (CSV)  
- Contains information about:
  - Order Date  
  - Region  
  - Category & Sub-Category  
  - Sales  
  - Profit  
  - Customer Segment  

## Technologies Used
- Python  
- Pandas  
- NumPy  

## Operations Performed
- Data loading and inspection  
- Data cleaning and preprocessing  
- Handling missing values  
- GroupBy operations for analysis  
- Sales and profit aggregation  
- Region-wise and category-wise analysis  
- Sorting and filtering data  

## Key Analysis Performed
- Total sales and profit calculation  
- Top-selling categories  
- Most profitable regions  
- Loss-making segments  
- Customer segment analysis  

## Key Insights
- Certain categories generate high sales but low profit  
- Some regions outperform others significantly  
- Discounts impact profitability  
- Identified best and worst performing segments  

## How to Run the Project
1. Install required libraries  
   pip install pandas numpy matplotlib seaborn  

2. Run the notebook  
   jupyter notebook  

## Conclusion
This project demonstrates how Pandas can be used to perform effective Exploratory Data Analysis (EDA) and extract valuable business insights from real-world datasets.
