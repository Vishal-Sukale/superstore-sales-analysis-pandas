# Superstore Sales Analysis (EDA) using Pandas

## Problem Statement
Retail businesses generate large volumes of transactional data, but extracting useful insights from this data is challenging. This project performs Exploratory Data Analysis (EDA) on the Superstore dataset to understand sales performance, profitability, and customer behavior.

## Objective
- Analyze sales and profit trends  
- Identify top-performing categories and regions  
- Detect loss-making areas  
- Generate insights to support business decisions  

## Dataset Description
- Dataset: Superstore Dataset  
- Format: CSV (tabular data)  

### Key Features:
- Order Date  
- Ship Date  
- Region  
- Category & Sub-Category  
- Sales  
- Profit  
- Quantity  
- Discount  
- Customer Segment  

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn
- 
## Data Analysis Steps
- Data loading and inspection  
- Handling missing values  
- Data cleaning and formatting  
- Exploratory Data Analysis (EDA)  
- GroupBy and aggregation  
- Sorting and filtering  

## Key Analysis Performed
- Total Sales & Total Profit  
- Region-wise Sales Analysis  
- Category & Sub-category performance  
- Customer Segment Analysis  
- Profit vs Discount relationship  
- Top-selling and loss-making products
- 
## Key Insights
- Some categories generate high sales but low profit  
- Discounts negatively impact profitability  
- Certain regions perform significantly better  
- Technology category often yields higher profit margins

This project demonstrates how Pandas can be used to analyze real-world retail data and extract meaningful business insights through EDA.
